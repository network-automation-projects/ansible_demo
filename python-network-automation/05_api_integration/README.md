# Module 05: API Integration

Integrating with REST APIs and source of truth systems (NetBox, Nautobot).

## Learning Objectives

- How to make HTTP requests (GET, POST, PUT, DELETE)
- How to use requests library effectively
- How to integrate with NetBox/Nautobot
- How to manage sessions and authentication
- How to handle API errors and retries

## Prerequisites

- Module 01: Core Fundamentals
- Module 03: File I/O Operations
- Basic understanding of REST APIs

## Concepts Covered

### Requests Library
- `requests.get()` - GET requests
- `requests.post()` - POST requests
- `requests.put()` - PUT requests
- `requests.delete()` - DELETE requests
- `requests.Session()` - Persistent sessions
- `requests.Response` - Response handling

### NetBox/Nautobot Integration
- Authentication with tokens
- Querying devices and IPs
- Creating and updating records
- Filtering and searching

## Use Cases

- Source of truth integration
- IPAM operations
- Device inventory management
- Dynamic device selection
