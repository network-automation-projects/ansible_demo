# Module 11: Cloud & Orchestration

Cloud and container automation (Docker, Kubernetes, AWS).

## Learning Objectives

- Docker API for container management
- Kubernetes client for cluster operations
- AWS Boto3 for cloud resources
- Container orchestration patterns

## Prerequisites

- Module 01: Core Fundamentals
- Module 05: API Integration

## Concepts Covered

- `docker.client.from_env/containers.run/images.pull` - Docker operations
- `kubernetes.client.CoreV1Api/list_pod_for_all_namespaces/load_kube_config` - K8s operations
- `boto3.client/resource/Session` - AWS operations

## Use Cases

- Container management
- Kubernetes automation
- Cloud network provisioning
