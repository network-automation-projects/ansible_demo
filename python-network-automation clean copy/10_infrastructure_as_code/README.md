# Module 10: Infrastructure as Code

Integrating Python with IaC tools (Terraform, Ansible).

## Learning Objectives

- Executing external tools with subprocess
- Ansible Python API integration
- Terraform integration
- CLI automation

## Prerequisites

- Module 01: Core Fundamentals
- Module 03: File I/O Operations

## Concepts Covered

- `subprocess.run/Popen/check_output` - Process execution
- `sys.argv/exit/stdin/stdout/stderr` - System interfaces
- Ansible Python API (inventory, loader, vars)
- Terraform integration patterns

## Use Cases

- Executing external tools
- CLI automation
- Hybrid Python/IaC workflows
