## Setup

### Required API Keys
Set the following environment variables:
- `ALPHAVANTAGE_API_KEY` - Get from https://www.alphavantage.co/support/#api-key
- `NEWS_API_KEY` - Get from https://newsapi.org
- `TWILIO_ACCOUNT_SID` - Get from https://www.twilio.com
- `TWILIO_AUTH_TOKEN` - Get from https://www.twilio.com
